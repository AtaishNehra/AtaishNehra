## 🧩 Key Projects

- [**Content Generation - Agentic AI**](https://github.com/AtaishNehra/content-gen)  
  An agentic workflow system that transforms long-form blog content into optimized social media posts with automated fact-checking, compliance review, and intelligent scheduling. Built with FastAPI, LangGraph, and OpenAI, the system extracts actionable insights, adapts content for each platform, validates facts using multiple providers, enforces regulatory compliance, and recommends optimal posting times. Features robust API endpoints, web UI, extensive test coverage, and production-ready orchestration for reliable, compliant, and high-quality content distribution.

- [**Multi-Agent Research System**](https://github.com/AtaishNehra/multi-agent-research-system)  
  Architected a multi-agent pipeline (Retriever, Summarizer, Synthesizer, Knowledge Base agents) leveraging FAISS, Hugging Face, and GPT-4o, delivering 90%+ factual accuracy and <30s end-to-end response time on enterprise-scale datasets.

- [**Twitch Games & Streams Recommender**](https://github.com/AtaishNehra/twitch-games-recommender)  
  Developed a content-based recommendation engine using SVD and ALS algorithms on Twitch API data, achieving a precision score of 9.1 for streamer/game clustering and user engagement optimization.

- [**Social Media Depression Analyser**](https://github.com/AtaishNehra/social-media-depression-analyser)  
  *IEEE Published*  
  Created an RNN-based NLP system for early detection of depressive language patterns in social media posts, with high recall for mental health risk indicators.

- [**Uber Data Analysis**](https://github.com/AtaishNehra/uber-data-analysis)  
  Analyzed large-scale Uber ride and allocation data using R, driving actionable urban mobility insights and optimization recommendations.

- [**Movie Reservation System**](https://github.com/AtaishNehra/movie-reservation-system)  
  Built an ETL-driven reservation system, transforming nested JSON to a normalized relational schema using MongoDB and Python.

- [**Milk Production Time-Series Analysis**](https://github.com/AtaishNehra/milk-production-timeseries-analysis)  
  Performed advanced forecasting and anomaly detection on milk production data using Python, enabling data-driven supply chain decisions.
